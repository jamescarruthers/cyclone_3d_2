# cyclone_3d_2

SkoolKit-generated starter disassembly for the ZX Spectrum game in this repo.

## Generated artifacts

- `cyclone.z80` - snapshot extracted from `cyclone.rzx`
- `cyclone.map` - execution map generated by replaying `cyclone.rzx`
- `cyclone.ctl` - control file generated from the snapshot plus the execution map
- `cyclone.skool` - initial SkoolKit disassembly generated from the control file

## Commands used

```sh
unzip -o Cyclone.tzx.zip -d /tmp/cyclone_tzx
rzxinfo.py --extract cyclone.rzx
mv cyclone.rzx.001.z80 cyclone.z80
rzxplay.py --no-screen --fps 0 --quiet --map cyclone.map cyclone.rzx
sna2ctl.py -m cyclone.map cyclone.z80 > cyclone.ctl
sna2skool.py -c cyclone.ctl cyclone.z80 > cyclone.skool
```
